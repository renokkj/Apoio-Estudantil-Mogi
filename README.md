# Apoio Estudantil Mogi

Protótipo de aplicação desenvolvida com Flutter e Dart com o objetivo de oferecer suporte acadêmico a estudantes, facilitando o acesso a informações e recursos educacionais.

## Objetivo do Projeto

O projeto foi criado com a proposta de centralizar funcionalidades acadêmicas em uma única aplicação, promovendo organização, acessibilidade e suporte aos estudantes.

## Funcionalidades

- Cadastro e gerenciamento de informações;
- Organização de dados acadêmicos;
- Navegação estruturada entre funcionalidades;
- Integração com banco de dados;
- Controle de fluxo da aplicação.

## Tecnologias Utilizadas

- Dart
- Flutter
- FlutterFlow (em partes do desenvolvimento)
- Git para versionamento

## Conceitos Aplicados

- Modelagem de dados
- Organização e estruturação de código
- Separação de responsabilidades
- Lógica de programação
- Integração entre interface e dados
